### NodeJs 版本 测试号推送消息卡片

#### 测试号地址

https://mp.weixin.qq.com/debug/cgi-bin/sandboxinfo?action=showinfo&t=sandbox/index

#### 使用说明

替换配置文件中测试号所需 appid，templateIdOne，secret，openId

#### 运行项目

```javascript
    npm i

    npm run dev
```
